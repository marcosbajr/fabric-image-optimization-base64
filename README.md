# fabric-image-optimization-base64

## 🇺🇸 English

An end-to-end image optimization pipeline built on Microsoft Fabric Lakehouse, transforming binary images into compressed and Power BI-ready Base64 strings with automatic resizing and block splitting.

## 🇧🇷 Português

Um pipeline completo de otimização de imagens construído no Lakehouse do Microsoft Fabric, transformando imagens binárias em strings Base64 comprimidas e prontas para uso no Power BI, com redimensionamento e divisão automática em blocos.
